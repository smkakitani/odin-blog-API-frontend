# Blog REST API 

This front-end connects to the Blog REST API, built with React + Vite. Read, write, and manage blog posts. Create and delete comments on posts.

Live demo: [https://blogapi-higurashi.netlify.app](https://blogapi-higurashi.netlify.app/)

This project was created following [The Odin Project - Project: Blog API](https://www.theodinproject.com/lessons/node-path-nodejs-blog-api) lesson.
